# Asset Library

Every image in this folder, categorized and described. Use this as your source of truth for which asset to pull when designing any section of the site. All images live under `yanik-design-handoff/images/` and can be referenced from HTML via absolute raw URLs:

```
https://raw.githubusercontent.com/guldanjaMAX/jamesguldan-site/main/yanik-design-handoff/images/FILENAME
```

## How to reference images in HTML

Use absolute raw GitHub URLs so they render anywhere:

```html
<img src="https://raw.githubusercontent.com/guldanjaMAX/jamesguldan-site/main/yanik-design-handoff/images/doodle-sloth-final.png" alt="Cebu the cosmic sloth">
```

If you are only editing the myth-storybook.html in this folder, the HTML already uses these absolute URLs. You do not need to change anything about the image references unless you are swapping one image for another.

## Usage rules

1. Do not upscale, restyle, or AI regenerate any of these images. Yanik's hand is his hand. Use what is here.
2. If you need a crop, crop. If you need a tint overlay, overlay. Do not edit the underlying illustration.
3. Celebrity photos are real people. Do not caption them with quotes they did not say. If you need placeholder quotes, mark them clearly so James can swap in real ones.
4. Tattoo photos are intimate. Use tastefully, never as decoration, always as meaning carrying proof of the myth.
