HOW TO USE THIS CATALOG
=======================

1. TO FIND ALL TESTIMONIALS FOR A SPECIFIC EVENT:
   Search for the event section name (e.g., "CAMP MAVERICK TESTIMONIALS")
   All testimonials for that event will follow in structured format

2. TO FIND A SPECIFIC PERSON'S TESTIMONIALS:
   Use Ctrl+F to search for their name (e.g., "NAME: John Ratliff")
   Multiple entries will show all their testimonials across events

3. TO EXTRACT A QUOTE FOR MARKETING:
   Find the person's name, note the event and year
   Copy the text in the QUOTE: field
   Reference the SOURCE FILE for verification

4. TO ANALYZE THEMES:
   See TESTIMONIALS_SUMMARY.txt for key themes organized by category
   Themes include: Connection, Impact, Business Growth, Experience Quality

5. TO TRACK REPEAT ATTENDEES:
   Look for people appearing in multiple events
   Top advocates: Shari Aldrich, Steve Young, John Ratliff, Jim Spano
